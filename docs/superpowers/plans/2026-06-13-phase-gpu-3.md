> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Подключить основной стек `science-pub` к существующим GPU endpoint'ам на `192.168.88.20` через LiteLLM, не ломая milestone 1 и не перенося inference-сервисы GPU-ноды в Docker.

**Architecture:** Backend и worker остаются на `sci_docker`, а LiteLLM становится маршрутизатором к внешним OpenAI-compatible endpoint'ам `qwen3.6` и `bge-m3` на GPU-ноде. В коде мы добавляем явный конфиг GPU-интеграции, переводим `LiteLLMProvider` из заглушки в рабочий HTTP-клиент и расширяем `/health` и `/config-check`, чтобы они показывали readiness и degraded-состояние внешних upstream'ов.

**Tech Stack:** FastAPI, Pydantic Settings, httpx, pytest, Docker Compose, LiteLLM, SSH

- [ ] **Step 3: Реализовать минимальный рабочий HTTP-клиент к LiteLLM**

```python
class LiteLLMProvider:
    ...
    async def generate(self, prompt: str, model: str | None = None) -> str:
        model_name = model or self.default_model
        if not model_name:
            raise ProviderNotReadyError("LiteLLM model is not configured yet.")

        payload = {
            "model": model_name,
            "messages": [{"role": "user", "content": prompt}],
        }

        try:
            async with httpx.AsyncClient(timeout=self.timeout) as client:
                response = await client.post(
                    f"{self.base_url}/chat/completions",
                    json=payload,
                )
                response.raise_for_status()
        except httpx.HTTPError as exc:
            raise ProviderNotReadyError(f"LiteLLM request failed: {exc}") from exc

        data = response.json()
        try:
            return data["choices"][0]["message"]["content"]
        except (KeyError, IndexError, TypeError) as exc:
            raise ProviderNotReadyError(
                "LiteLLM returned an unexpected response payload."
            ) from exc
```

```python
@router.get("/config-check", response_model=ConfigCheckResponse)
async def config_check(
    settings: AppSettings = Depends(get_settings_dependency),
    health_service: HealthService = Depends(get_health_service),
) -> ConfigCheckResponse:
    checks = await health_service.config_checks()
    warnings = [
        settings.health_warning_litellm,
        settings.health_warning_gpu_partial,
    ]
    return ConfigCheckResponse(
        valid=all(item.ok or not item.required for item in checks.values()),
        warnings=warnings,
        config={
            "environment": settings.environment,
            "litellm_model": settings.litellm_model,
            "gpu_node_host": settings.gpu_node_host,
        },
        checks=checks,
    )
```
